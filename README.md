## Options

```vim
" default options.
let g:github = {
  \ 'theme': 'dark'
  \ }
```

|Key|Type|Deafult|Description|
|---|---|---|---|
|`theme`|string|`'dark'`|The flavor of this colorscheme. Available: `'dark'`, `'light'`, `'dark_dimmed'`|
